Módulo 02
📄

Planificación del EID en LaTeX

Objetivo Mineduc · Planificación · .tex

Aprenderás a crear una planificación profesional del EID partiendo de los objetivos del Mineduc, usando directamente LaTeX desde cero.

  • Análisis del objetivo general del EID y los lineamientos Mineduc para la sesión
  • Planificación en .tex — ejemplo con modelo de crecimiento poblacional
  • Plantilla propia — usarás una página elaborada por el profe que entrega la estructura LaTeX lista
LaTeX EID Mineduc
Módulo 03
💻

Guías EDO con IA: .tex y HTML pro

Guía · Claude AI · GitHub Pages

El módulo más completo: crearás la guía de trabajo en LaTeX, la transformarás en HTML básico y luego la potenciarás con IA hasta publicarla en la web.

  • Guía en .tex — estructura y estética básica de LaTeX con apoyo de ChatGPT gratuito
  • Export HTML — desde la misma página del profe se genera el HTML base de la guía
  • Prompt .md en Claude — un prompt muy detallado potencia el HTML básico a una versión pro
  • GitHub Pages — guardas el archivo, lo publicas y obtienes la URL para insertar en el .tex
Claude AI LaTeX HTML GitHub Pages

Formulario de cierre personalizado

Reflexión · Evaluación · Cierre de clase

Módulo opcional en desarrollo: aprenderás a crear un formulario personalizado que tus estudiantes completarán al finalizar la clase del EID.

  • Espacio de comentarios — los estudiantes podrán compartir cómo fue su experiencia de clase
  • Herramientas y plataforma por confirmar
