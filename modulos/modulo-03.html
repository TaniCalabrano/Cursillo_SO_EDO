Módulo 03 · LaTeX · HTML · Claude AI · GitHub Pages

Creación del recurso didáctico: Guía .tex y complemento digital HTML

Aprenderás a crear la guía de trabajo en LaTeX usando la herramienta del profe, exportarla como HTML básico y transformarla en una versión profesional con Claude. Finalmente la publicarás en GitHub Pages.

1
Resumen del flujo de trabajo
🎯
¿Qué crearás en este módulo?

Al finalizar habrás producido dos entregables complementarios para tu EID:

📄
Guía en LaTeX (.tex)

Documento imprimible de alta calidad tipográfica. Se compila en Overleaf con logos institucionales y produce un PDF listo para imprimir o compartir.

LaTeX Overleaf PDF
🌐
Guía digital HTML (pro)

Versión web interactiva con modo Docente y Estudiante, ecuaciones con MathJax, gráficos y formulario de cierre. Se publica en GitHub Pages.

HTML Claude AI GitHub Pages
🗺️
Pasos del módulo
🔧
Paso 1 · Generador de guías
Herramienta propia

Usas la herramienta Generador de Guías (incluida en este módulo) para completar el formulario con los datos de tu guía. Desde ahí obtienes el código .tex y el HTML básico.

☁️
Paso 2 · Compilar en Overleaf con logos
Overleaf PRO

No subas los logos directamente. El profe compartirá a cada grupo un proyecto Overleaf con funciones PRO que ya tiene la estructura lista, las imágenes institucionales precargadas y el preámbulo correcto.

Tu tarea: pegar el código .tex generado en ese proyecto, reemplazar solo el preámbulo y compilar.

🤖
Paso 3 · Transformar HTML básico → HTML pro con Claude
Claude AI

Tomas el HTML básico y se lo entregas a Claude junto con el prompt .md descargable en este módulo. Claude transformará el HTML en una versión profesional con modo Docente/Estudiante, MathJax y exportación de respuestas.

🚀
Paso 4 · Publicar en GitHub Pages e insertar URL
GitHub Pages

Subes el HTML pro a GitHub, activas GitHub Pages y obtienes una URL pública. Esa URL la pegas en tu .tex como enlace o código QR para que los estudiantes accedan desde el PDF impreso.

Compilar en Overleaf con logos institucionales
⚠️
No conviene subir los logos directamente desde el generador. Las imágenes institucionales están precargadas en un proyecto Overleaf con funciones PRO que el profe compartirá a cada grupo. Esto garantiza logos correctos en resolución y posición adecuada.
Overleaf Instrucciones para compilar tu guía
  1. 1Espera a que el profe comparta el enlace al proyecto Overleaf de tu grupo. Recibirás un correo con acceso como colaborador.
  2. 2Abre el proyecto compartido. Verás el archivo main.tex con el preámbulo institucional, logos y paquetes listos.
  3. 3Genera el código .tex con el Generador. Copia solo el cuerpo del documento (desde \begin{document} hasta \end{document}).
  4. 4No modifiques el preámbulo del archivo compartido. Solo pega tu contenido dentro del cuerpo.
  5. 5Presiona Recompilar en Overleaf. El PDF se generará con logos y formato institucional.
  6. 6Descarga el PDF final. También puedes compartir el link de vista previa de Overleaf.
ℹ️
Funciones PRO de Overleaf: compilación rápida, historial ilimitado, colaboración en tiempo real y soporte para paquetes avanzados como tikz, pgfplots y fuentes personalizadas.

5
Publicar en GitHub Pages e insertar URL en el .tex
🚀
¿Por qué GitHub Pages?

Te permite publicar archivos HTML de forma gratuita y obtener una URL pública permanente como https://usuario.github.io/repo/guia.html para insertar en tu .tex.

GitHub Pasos para publicar y obtener la URL
  1. 1Si no tienes cuenta, créala en github.com. Es gratis.
  2. 2Crea un repositorio nuevo (ej. guia-edo). Márcalo como público.
  3. 3Sube tu guia_interactiva.html con el botón "Add file → Upload files".
  4. 4Ve a Settings → Pages. En "Source" selecciona Deploy from a branch, elige main y carpeta / (root). Guarda.
  5. 5Espera 1-2 minutos. Refresca y aparecerá tu URL: https://tu-usuario.github.io/guia-edo/guia_interactiva.html
  6. 6Copia esa URL y pégala en tu .tex como hipervínculo o código QR.
📎
Insertar la URL en el .tex

Con el paquete hyperref (ya incluido en el proyecto Overleaf compartido):

fragmento .tex
Simplemente tienes que reemplazar "https://example.com" por tu URL en el bloque \begin{tikzpicture} \node[inner sep=0] (qr) { \color[HTML]{1F3554} \qrcode[height=3.5cm]{https://example.com} }; \node at (qr.center) { \includegraphics[width=1.5cm]{LOG_SEMINARIO.png} }; \node[below=3pt of qr,align=center] { \scriptsize Guía interactiva en línea }; \end{tikzpicture}
